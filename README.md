# Crypto ToDo DApp

Децентрализованное приложение для управления задачами с хранением данных в блокчейне через смарт-контракт на Solidity.

---

## Описание проекта

Проект демонстрирует применение технологии блокчейн для постоянного и прозрачного хранения пользовательских данных. Вместо традиционной базы данных или localStorage все задачи записываются в смарт-контракт, развёрнутый в сети Ethereum. Каждая операция (добавление, выполнение, удаление задачи) является блокчейн-транзакцией, подписанной через MetaMask.

## Смарт-контракт

Файл: `TodoList.sol`

### Структура задачи

```solidity
struct Task {
    uint    id;
    string  title;
    string  description;
    string  category;
    string  date;
    bool    isDone;
    bool    exists;
    address owner;
}
```

### Функции контракта

| Функция | Тип | Описание |
|---|---|---|
| `addTask(title, description, category, date)` | транзакция | Добавить новую задачу |
| `completeTask(id)` | транзакция | Отметить задачу выполненной |
| `deleteTask(id)` | транзакция | Удалить задачу |
| `getTask(id)` | view (чтение) | Получить данные задачи по id |
| `getMyTaskIds()` | view (чтение) | Получить все id задач текущего пользователя |

### Особенности реализации

- Каждый пользователь видит только свои задачи — изоляция по адресу (`msg.sender`)
- Модификатор `taskExists` защищает от обращения к удалённым задачам
- Модификатор `onlyOwner` запрещает изменять чужие задачи
- События (`events`) позволяют отслеживать все операции в блокчейне

## Установка и запуск

### 1. Установить MetaMask

Установи расширение [MetaMask](https://metamask.io) в браузер и создай кошелёк.

### 2. Получить тестовый ETH

Переключись на тестовую сеть **Sepolia** в MetaMask и получи бесплатный ETH через faucet:
- [sepoliafaucet.com](https://sepoliafaucet.com)

### 3. Задеплоить смарт-контракт в Remix IDE

1. Открой [remix.ethereum.org](https://remix.ethereum.org)
2. Создай файл `TodoList.sol` и вставь содержимое из проекта
3. Вкладка **Solidity Compiler** → выбери версию `0.8.x` → **Compile**
4. Вкладка **Deploy & Run Transactions**:
   - Environment: `Browser Extension — MetaMask`
   - Нажми **Deploy** → подтверди транзакцию в MetaMask
5. Скопируй адрес задеплоенного контракта

### 4. Вставить адрес контракта

Открой `app.js` и замени значение в первой строке:

```js
const CONTRACT_ADDRESS = "0xАдресТвоегоКонтракта";
```

### 5. Открыть приложение

Открой `index.html` в браузере — MetaMask подключится автоматически.

## Как работает приложение

1. При загрузке страницы `app.js` подключается к MetaMask и создаёт экземпляр контракта через ethers.js
2. Вызывается `getMyTaskIds()` — получаем список id задач текущего пользователя
3. Для каждого id вызывается `getTask(id)` — загружаются данные задачи из блокчейна
4. При добавлении задачи отправляется транзакция `addTask()` — данные записываются в блокчейн навсегда
5. Каждое действие (выполнить, удалить) также является транзакцией, требующей подписи в MetaMask

---

## Функциональность интерфейса

- Добавление задачи с названием, описанием, категорией и датой
- Три вкладки: **В работе**, **Выполнено**, **Просрочено**
- Поиск по названию задачи
- Фильтрация по категории
- Сортировка по дате (ближайшие / дальние)
- Отметить задачу выполненной
- Удалить задачу

---

## Проверка работы через Remix

После деплоя и добавления задач можно проверить данные прямо в Remix:

- **`getMyTaskIds`** → Call — возвращает массив id твоих задач, например `[0, 1]`
- **`getTask`** → ввести id (например `0`) → Call — возвращает все поля задачи
- **`taskCount`** → Call — общее количество задач в контракте
